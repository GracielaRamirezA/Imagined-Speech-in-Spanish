# EEG Dataset and Transformer-Based Classification Framework

All the material shared in this repository, including datasets and source code, may be freely used for research and academic purposes provided that the corresponding dataset publication article is properly cited.

## Repository Structure

This repository contains the complete pipeline used for EEG acquisition, preprocessing, segmentation, classification, and additional signal analysis.

### 1. Dataset Raw

This folder contains the raw EEG recordings acquired directly from the hardware setup before preprocessing.

The acquisition system used was:

* OpenBCI Ultra Cortex Mark IV headset
* OpenBCI Cyton + Daisy board configuration

The raw recordings correspond to the original EEG signals collected during the experiments.

---

### 2. Experiment

`Experiment` is the first code that should be executed.

This script is responsible for:

* EEG signal acquisition
* Communication with the OpenBCI hardware
* Experimental control during recording sessions
* Saving the raw EEG recordings

The generated files are later used for preprocessing and segmentation.

---

### 3. Dataset Processed

This folder contains the processed and segmented EEG recordings generated after preprocessing the raw signals.

These files are the ones used for training and evaluating the classification models.

---

### 4. Processing and Segmentation

This script performs:

* EEG preprocessing
* Filtering
* Segmentation
* Data organization for classification tasks

To execute this code:

* The raw dataset files must be located in the same directory as the script.

The output generated by this step corresponds to the files stored in `Dataset Processed`.

---

### 5. Transformer_hi

`Transformer_hi` performs:

* Model training
* Validation
* Testing of the implemented Transformer-based architecture using the processed EEG dataset.

To execute this code:

* The processed dataset files must be located in the same folder as the script.

This implementation is based on the following work:

@inproceedings{gallo2024thinking,
  title={Thinking is Like Processing a Sequence of Spatial and Temporal Words},
  author={Ignazio Gallo, Silvia Corchs},
  booktitle={2024 International Joint Conference on Neural Networks (IJCNN)},
  year={2024},
  publisher={{IEEE}},
  isbn={978-8-3503-5931-2},
}

---

### 6. Ablation Study and Signal Quality Analysis

These scripts were used to:

* Perform channel ablation experiments
* Evaluate processed EEG signal characteristics
* Analyze inter-subject variability
* Study the relationship between signal characteristics and classifier performance

To execute these scripts:

* The processed dataset files must be located in the same folder as the scripts.

---

## General Workflow

1. Acquire EEG signals using `Experiment`
2. Store recordings inside `Dataset Raw`
3. Run preprocessing and segmentation `Processing_and_Segmentation`
4. Generate processed files inside `Dataset Processed`
5. Train and evaluate the model using `Transformer_hi`
6. Perform additional analyses such as ablation and signal quality studies
